## Running locally

1. Clone this repository
2. Crate a new Supabase Collection
3. Crate an `.env` File with your `SUPABASE_URL` and `SUPABASE_KEY` Environment Variables
4. Install with `yarn`
5. Run with `yarn run`
6. Open http://localhost:3000/ in your browser
